# LumioSmartHome
sebuah Website Berbasis IoT (ESP32) Untuk mengontrol jarak jauh sebuah lampu maupun peralatan elektronik lain di rumah. 

# FITUR :

# 1. ON/OFF (nyala/mati).
Kontrol penuh nyala atau mati n         Lampu/Alat elektronik lain. bisa        perdevice maupun langsung semua         (ALL ON/OFF). 

# 2. Rooms. 
untuk mengelompokkan lampu atau         alat elektronik lainnya bedasarkan      rooms/ruangannya. dengan begitu         bisa melakukan ON/OFF sekaligus         semua device di rooms tersebut. 

# 3. Schedule. 
melakukan ON/OFF pada lampu/device      elektronik lainnya menggunakan          jadwal yang sudah di tentukan. 

# 4. Log.
merekam semua log aktivitas yang        dilakukan di web ini. seperti           melakukan aktivitas ON/OFF, dll.
